# рџ“Њ Overview

DealHunt is a full-stack web application that enables users to compare product prices across multiple e-commerce platforms from a single interface.

Instead of manually visiting multiple websites, users can search using a product name, SKU, or product URL and instantly view live prices, identify the best deal, monitor price history, and receive price-drop alerts.
