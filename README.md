# 📌 Overview

DealHunt is a full-stack web application that enables users to compare product prices across multiple e-commerce platforms from a single interface.

Instead of manually visiting multiple websites, users can search using a product name, SKU, or product URL and instantly view live prices, identify the best deal, monitor price history, and receive price-drop alerts.

# 📊 Core Modules

## 🔍 Search & Price Comparison

Search products using:

- Product Name
- SKU
- Product URL

The application collects prices from multiple stores and displays the lowest available price.

---

## 📈 Price Analytics

- Lowest Recorded Price
- Highest Recorded Price
- Average Price
- Price Trend Graph

---

## ❤️ Watchlist

Users can save products for future tracking.

The watchlist stores:

- Entry Price
- Current Price
- Lowest Store
- Percentage Price Drop

---

## 🔔 Smart Alerts

Users can configure a minimum and maximum target price.

Whenever the product price falls within the specified range, a notification is generated.
